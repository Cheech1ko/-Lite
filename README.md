Для полной работы с бэкендом необходимо локально запустить сервер:

1. Скачать проект: git clone https://github.com/Cheech1ko/-Lite.git
2. Запустить бэкенд: cd backend && npm install && node server.js
3. Запустить фронтенд: cd frontend && npm install && npm start
